# מחשבון ריבית דריבית ויזואלי - Compound Interest Simulator

זהו פרויקט הסיום שלי לקורס **Practical AI: Chatbots as Research Copilots**.
האפליקציה פותחה בשיטת **Vibe Coding** תוך שימוש בכלי בינה מלאכותית גנרטיבית.

## אודות הפרויקט
האפליקציה מאפשרת למשתמשים לבצע סימולציה של צמיחת הון לאורך זמן באמצעות חישוב ריבית דריבית. היא נועדה להנגיש תכנון פיננסי בצורה ויזואלית ופשוטה.

### תכונות מרכזיות:
* **חישוב דינמי:** הזנת סכום ראשוני, הפקדה חודשית, אחוז תשואה וזמן.
* **ויזואליזציה:** גרף צמיחה אינטראקטיבי המבוסס על ספריית `Chart.js`.
* **עיצוב מודרני:** ממשק רספונסיבי מותאם לעברית (RTL).

## טכנולוגיות וכלים
* **פיתוח:** AntiGravity (IDE) בסיוע Gemini/Claude.
* **שפות:** HTML5, CSS3, JavaScript.
* **גרפיקה:** Chart.js.
* **אחסון ופריסה:** GitHub & GitHub Pages.

## תהליך העבודה (Workflow)
1. **אפיון:** יצירת מסמך דרישות (PRD) בעזרת AI.
2. **קידוד:** פיתוח איטרטיבי ב-AntiGravity במצב Agent.
3. **בדיקות:** וולידציה של לוגיקת החישוב מול מחשבונים פיננסיים קיימים.
4. **Deployment:** העלאה ל-GitHub והפעלה כאתר חי דרך GitHub Pages.
