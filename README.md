# מחשבון ריבית דריבית חכם - Compound Interest Simulator (v2.0)

זהו פרויקט הסיום המשודרג לקורס **Practical AI: Chatbots as Research Copilots**.
האפליקציה פותחה בשיטת **Vibe Coding** תוך שימוש בכלי בינה מלאכותית גנרטיבית, ועברה התאמות אישיות לשיפור חווית המשתמש (UX).

## עדכונים בגרסה 2.0 (בעקבות פידבק המרצה)
בעקבות הערות המרצה, בוצעו השדרוגים הבאים כדי להפוך את הכלי למוצר נגיש ומקצועי יותר:
* **Onboarding (תהליך קליטה):** הוספת מסך פתיחה אינטראקטיבי המסביר למשתמש את חשיבות הריבית דריבית ואת אופן השימוש בכלי.
* **ממשק משתמש (UI) משופר:** עיצוב מחדש של הממשק עם פלטת צבעים מודרנית, רכיבים מעוגלים וצללים לשיפור האסתטיקה.
* **נגישות ומידע (Tooltips):** הוספת הסברים (Info Icons) לצד כל שדה קלט כדי לסייע למשתמשים להבין את המונחים הפיננסיים.

## אודות הפרויקט
האפליקציה מאפשרת סימולציה של צמיחת הון לאורך זמן. המשתמש מזין סכום ראשוני, הפקדה חודשית, אחוז תשואה וזמן, ומקבל פלט מספרי לצד גרף צמיחה דינמי.

### טכנולוגיות וכלים
* **פיתוח:** AntiGravity (IDE) בסיוע Gemini.
* **שפות:** HTML5, CSS3, JavaScript.
* **גרפיקה:** Chart.js (ויזואליזציה של נתונים).
* **אחסון ופריסה:** GitHub & GitHub Pages.

## תהליך העבודה (Workflow)
1. **אפיון:** יצירת מסמך דרישות (PRD) מעמיק הכולל פתרון להערות המרצה.
2. **קידוד:** פיתוח איטרטיבי ב-AntiGravity במצב Agent.
3. **בדיקות:** וולידציה של לוגיקת החישוב מול מודלים פיננסיים.
4. **Deployment:** העלאה ל-GitHub ופריסה דרך GitHub Pages כאתר חי.
